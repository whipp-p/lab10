Запустите Main.java — демонстрация всех сценариев в консоли.
UML-диаграммы (вариантов использования, активностей, последовательностей) доступны в Visual Paradigm.
Описание модели, основные классы:

Faculty — факультет, имеет декана и кафедры
Cafedra — кафедра, содержит научных сотрудников
ScientificWorkers — научный сотрудник (наследует Employee)
Decan, Lecturer, Secretary — роли сотрудников
Course — учебный курс
Project — научный проект
Partaking — участие в проекте (связка)
